Mode 1: Default (Standard RDF Prefixes)

Empty tag loads standard RDF prefixes: dc, foaf, rdf, rdfs, schema, skos, wp, owl2

Mode 2: Empty (User Adds Everything)

Starts completely blank. User must add all prefixes manually. Perfect for domain-specific apps.

Mode 3: Declarative (Custom Prefix Set)

Pre-configured with specific prefixes. Note 'bartism' and 'simpson' are pre-selected.

Code Examples

Default Mode:

<prefixes-form></prefixes-form>

Empty Mode:

<prefixes-form empty></prefixes-form>

Declarative Mode:

<prefixes-form>
  <prefix-entry prefix="homerism" expansion="https://why.eye.ough.ta/" />
  <prefix-entry prefix="bartism" expansion="https://eat.my/shorts#" selected />
  <prefix-entry prefix="simpson" expansion="http://springfield.tv/family#" 
                tooltip="The Simpson family" selected />
</prefixes-form>
